TEST SUITE E2E CON SELENIUM WEBDRIVER

* OPCION ELEGIDA: #2

-   Se diseñaron, desarrollaron y ejecutaron casos de prueba para el sitio https://www.saucedemo.com/
    El proyecto fue estructurado de acuerdo al patrón PAGE OBJECT MODEL (POM), para optimizar el código
    y evitar la duplicación del mismo.

-   La test suite E2E se ecuentra en el siguiente path: "org/example/SauceDemo/tests/E2E/E2ETests.java",
    desde allí se puede ejecutar la suite completa.
    Una vez ejecutados los tests, se autogenera la carpeta "TestsScreenshots", el cual contiene todas las
    capturas de pantalla tomadas a lo largo de la testsuite, y el archivo "report.html", en el cual se puede
    observar de forma gráfica el resumen de la suite ejecutada.

-   Por otra parte, se pueden ver los test organizados según la funcionalidad en las carpetas "HomePage", "InventoryPage"
    y "FinalCheckOutPage".
